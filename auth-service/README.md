# The Restaurant Engine - Auth Service 🍽️

このプロジェクトは、飲食店の管理を効率化するためのシステムです。
現在は、**FastAPI** を使用した「認証およびユーザー管理サービス」の構築に注力しています。

## 🎯 今回のフェーズ
- **Python / FastAPI** を使用した堅牢なAPIの開発
- Pydanticによる厳密なデータバリデーション
- AWS (Lambda/DynamoDB) へのデプロイを視野に入れた設計

## 🏗️ 開発中の機能
- [x] プロジェクト構造のセットアップ
- [x] 基本的なAPIエンドポイントの作成
- [ ] ユーザー登録バリデーションの追加
- [ ] JWT（JSON Web Token）による認証の実装

## 🛠️ 技術スタック
- **Backend**: Python 3.10+ / FastAPI
- **Web Server**: Uvicorn
- **Validation**: Pydantic
